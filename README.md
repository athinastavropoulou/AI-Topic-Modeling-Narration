# Automated Topic Modeling & AI Audio Narration 🤖🎙️

## Overview
This project implements an advanced NLP pipeline that automatically extracts key topics from text documents and generates an AI-powered audio narration of the findings.

## Key Features
* **Topic Extraction:** Uses `tomotopy` for **LDA (Latent Dirichlet Allocation)**.
* **Text Processing:** Leverages `NLTK` for professional tokenization and noise reduction.
* **Generative AI Summary:** Connects to **Gemini 2.5 Flash API** for thematic summarization.
* **AI Narration (TTS):** Produces high-quality `.wav` audio files using Google's TTS capabilities.

## Tech Stack
* Python, Gemini API, tomotopy, nltk.
